Overview

What is this?

This documentation describes the JSON5 data structure used to store real-time vehicle positions from the Hungarian State Railways (M√ÅV) system. The data includes trains, trams, buses, and other public transportation vehicles across Hungary.

Data Source

emma.mav.hu/otp2-backend

Key Features

  • Real-time GPS coordinates
  • Schedule and delay information
  • Route and stop information
  • Multiple transport modes

Field Reference

Metadata Fields

Field Type Description
fetch_time string ISO 8601 timestamp when data was fetched (UTC)
source string Description of the data source
api_endpoint string URL of the API endpoint used
vehicle_count number Total number of vehicles in this data set

Vehicle Position Fields

Field Type Description
vehicleId string Unique identifier for the vehicle
lat number Latitude coordinate (WGS84)
lon number Longitude coordinate (WGS84)
heading number Direction of travel in degrees (0-360)
label string Human-readable vehicle identifier (e.g., "IC 123")
lastUpdated number Unix timestamp of last position update
speed number Current speed in km/h

Transport Modes

RAIL

Regular passenger trains

SUBURBAN_RAILWAY

S-Bahn style suburban trains

TRAMTRAIN

Tram-train hybrid vehicles

COACH

Long-distance bus services

RAIL_REPLACEMENT_BUS

Buses replacing train services

Stop Relationship Status

IN_TRANSIT_TO

Vehicle is traveling toward the next stop

STOPPED_AT

Vehicle is currently stopped at a station
